# Әділзаң site

Это простая статическая версия сайта для Vercel.

Перед запуском откройте `index.html` и замените:
- `77000000000` на WhatsApp номер без плюса;
- `+7 700 000 00 00` на телефон;
- `https://instagram.com/` на ссылку Instagram;
- `info@adilzan.kz` на email;
- `г. Астана, адрес офиса` на адрес.

На Vercel настройки:
- Build Command: `npm run build`
- Output Directory: `dist`
